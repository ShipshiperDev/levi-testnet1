# LEVI — Onchain Research Powerhouse

**the intelligence layer for the tempo ecosystem.**

levi is an autonomous onchain and offchain research engine built natively on the **tempo network**. it maps trending narratives, researches social signals on **x (twitter)**, and tracks influencer/token/nft alpha before it reaches the mainstream.

## the loop
levi operates on a self-sustaining autonomous economic framework:
**launch → earn → reinvest → scale**

using **machine payments protocol (mpp)** and **x402**, levi autonomously pays for its own compute, data oracles, and infrastructure, closing the loop between intelligence and execution.

## core features
- **social narrative intelligence**: real-time mapping of x (twitter) influencer clusters and trending narratives.
- **narrative tracking**: identifying what is actually working across the tempo network (tokens, nfts, liquidity).
- **autonomous research agents**: agents that earn and reinvest their own revenue to scale monitoring depth.
- **verifiable alpha**: every signal is traceable to raw chain behaviour or social fingerprints.

## technology stack
- **rail**: tempo network
- **logic**: mpp + x402
- **intelligence**: proprietary neural pattern recognition
- **frontend**: next.js + cinematic html5/css3 visuals

## getting started

first, install dependencies:
```bash
npm install
```

run the development server:
```bash
npm run dev
```

open [http://localhost:3000](http://localhost:3000) to view the terminal and research dashboard.

## cinematic ad
to view the cinematic launch advertisement, open `levi-ad.html` in a browser.
